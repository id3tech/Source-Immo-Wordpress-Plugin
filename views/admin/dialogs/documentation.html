Templates override

Vous pouvez remplacer la plupart des éléments de mise en page en incluant un dossier spécial dans le répertoire du thème actif. Le nom du dossier doit être source-immo et le modèle que vous souhaitez remplacer doit respecter le chemin de répertoire correspondant à ce plugin.

Exemple: vous souhaitez remplacer la disposition de la carte dans la liste des courtiers. Dans le répertoire du thème actif, ajoutez le fichier source-immo/views/list/brokers/standard/item-standard.php et bon codage.

Petit conseil, pour vous éviter les casse-têtes, copiez le fichier d'origine dans votre répertoire de thème et modifiez-le.

You can overrides most of the items layout by including a special folder in the active theme directory. The folder name must be source-immo and the template you wish to overrides must respect the directory path from this plugin.

Example: You want to overrides the list item layout for the brokers. In the active theme directory, add the file source-immo/views/list/brokers/standard/item-standard.php and happy coding.

Small advice, to avoid most trouble, copy the original file to your theme directory and modify it.
